Memoization and Why Does It Feel So Good to Optimize?

Apr 18th, 2014

Recently I was faced with a challenge from Project Euler and asked how I would best optimize the solution I wrote. Thinking for a little bit, I made a few changes, but none were as interesting as as term that I had never heard before, but understood the concept: Memoization.

Memoization is the concept of storing a data in memory that you may reference again at a later time, instead of calling a function with parameters you have already passed it previously. I’ll show you how we can use memoization and a little bit of logic to optimize a problem from Project Euler.
